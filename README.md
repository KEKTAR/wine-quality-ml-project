# Wine Quality Analysis

## Proje Amacı
Bu projede Wine Quality veri seti kullanılarak keşifçi veri analizi (EDA),
lineer regresyon ve sınıflandırma modelleri uygulanmıştır.

## Veri Seti
Wine Quality (Red) veri seti kullanılmıştır.
Hedef değişken, şarap kalitesinin iyi veya kötü olmasıdır.

## Kullanılan Yöntemler
- Keşifçi Veri Analizi (EDA)
- Lineer Regresyon (Alcohol vs Density)
- Logistic Regression
- K-Nearest Neighbors (KNN)
- Support Vector Machines (SVM)
- Decision Tree

## Sonuçlar
Modeller Accuracy, Precision, Recall ve F1-Score metrikleri ile
değerlendirilmiştir.
